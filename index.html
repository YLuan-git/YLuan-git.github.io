About Me

As a biostatistician, I have broadened my work to combine statistical innovation, biomedical application, and computational implementation.

I received a PhD in Epidemiology with a focus on Biostatistics from Indiana University, where I was advised by Dr. Carmen Tekwe. Following that, I spent two year as a postdoctoral research fellow in the School of Public Health at Indiana University in Dr. Carmen Tekwe and Dr. Roger Zoh's group.

My undergraduate degree was from University of Minnesota, where I received a Bachelor degree in Statistics.

Software

I develop R packages and software tools to support biostatistical research and make advanced methods accessible to the broader research community.

MECfda: Scalar-on-Function Regression with Measurement Error Correction

MECfda solves scalar-on-function linear models, including generalized linear mixed effect model and quantile linear regression model, and bias correction estimation methods due to measurement error.

The package implements advanced statistical methods for handling measurement error in functional data analysis. Key features include:

  • Scalar-on-function linear regression models
  • Generalized linear mixed effect models
  • Quantile linear regression models
  • Bias correction methods for measurement error

Details about the measurement error bias correction methods can be found in: Luan et al. (2023) doi:10.48550/arXiv.2305.12624, Tekwe et al. (2022) doi:10.1093/biostatistics/kxac017, Zhang et al. (2023) doi:10.5705/ss.202021.0246, Tekwe et al. (2019) doi:10.1002/sim.8179.

The package can be installed using

install.packages("MECfda")

Software Under Development

QuantileSIMEX

QuantileSIMEX is an R package for quantile functional data analysis with measurement error using the SIMEX method. This package provides tools for correcting bias in functional quantile regression models when covariates are measured with error.

Key features include:

  • SIMEX-based bias correction for functional quantile regression
  • Support for both functional and scalar error-prone covariates
  • Flexible quantile specification
  • Comprehensive simulation and diagnostic tools

Package currently in development. Expected release: 2025

Teaching Philosophy

My teaching approach emphasizes the practical application of statistical methods to real-world biomedical problems. I believe in making complex statistical concepts accessible through hands-on examples and encouraging students to think critically about data analysis challenges. I strive to create an inclusive learning environment where students feel comfortable asking questions and exploring statistical concepts deeply.
